AI Automation Engineer

Location: Remote Type: Full-Time / Contract

Overview

We are seeking an AI Automation Engineer to implement intelligent workflow systems powered by AI agents. This role focuses on building automation infrastructure that connects communication channels, scheduling systems, and operational workflows into a cohesive autonomous system.

You will be responsible for translating system architecture into production-ready automation pipelines.

Responsibilities

  • Build production-grade automation pipelines for AI reception workflows
  • Develop agent-based logic for lead qualification, patient triage, and appointment scheduling
  • Implement integrations with third-party APIs, communication platforms, and CRMs
  • Design workflow orchestration across messaging, web forms, and telephony systems
  • Develop data pipelines that support AI-driven decision processes
  • Monitor automation performance and optimize workflow latency and reliability
  • Implement logging, error handling, and system monitoring across automation pipelines

Requirements

  • Strong experience with API integrations and webhook-based systems
  • Experience with automation frameworks (n8n, Temporal, Airflow, or similar)
  • Familiarity with scripting languages such as Python or JavaScript
  • Understanding of asynchronous workflows and event-driven systems
  • Experience working with LLM APIs and AI toolchains
  • Knowledge of backend development and automation infrastructure
